node.js 버전 확인

npm 버전 확인
cmd 창에서 npm -v 명령어로 확인합니다.

package.json 파일 만들기
모듈을 관리하는 설정파일
/*
nodejs가 필요한 것들을 준비

*/

npm init -y : 기본값을 기준으로 바로 생성
